# 形态键列表及说明

形态键是用于变形3D模型形状的功能。Menno模型共实现了606个形态键，可以精细调整角色的表情和体型。

## 形态键概述

### 面部形态键（共564个）

| 类别 | 数量 | 说明 |
|---------|-----|------|
| VRC | 18个 | VRChat用形态键（v_aa、v_ch、v_dd等语音识别用和视线控制用） |
| EYE | 51个 | 眼睛表情控制（闭眼、笑眼、吊眼、下垂眼、半眯眼等） |
| EYE_EX | 3个 | 眼睛着色器效果（Shader_ON、Shader_Big、Grabpass_ON等） |
| EYE_Action | 7个 | 眼睛特殊效果（眼泪、爱心、旋转、震惊、星星等） |
| EYE_Option | 67个 | 眼睛细节调整（光、黑眼圈、位置、大小、间距等） |
| PUPIL_Option | 35个 | 瞳孔调整（白瞳孔、收缩、扩张、大小、前后移动等） |
| IRIS_Option | 18个 | 眼白部分调整（模糊、大小、前后移动等） |
| EYELID_Option | 74个 | 眼睑调整（前端、末端、上下左右的精细移动等） |
| EYELASH | 12个 | 睫毛表情（细眼、睡眼、温和眼等） |
| EYELASH_Option | 48个 | 睫毛调整（上下睫毛的粗细、长度、位置等） |
| EYEBROW | 48个 | 眉毛表情（上下、微笑、愤怒、困惑、各部位的移动等） |
| EYEBROW_Option | 10个 | 眉毛调整（粗细、形状、位置等） |
| MOUTH | 53个 | 口部表情（a、i、u、e、o、微笑、咧嘴、三角形、ω等各种形状） |
| MOUTH_Option | 8个 | 口部调整（尖牙、前后位置、脸颊等） |
| TONGUE | 12个 | 舌头表情（伸出、弯曲、前后上下移动等） |
| TONGUE_Option | 6个 | 舌头调整（细、粗、平、厚、长度等） |
| TEETH_Option | 15个 | 牙齿调整（尖牙、锯齿状、上下位置等） |
| FACE | 6个 | 整脸效果（脸红、震惊、皱纹等） |
| NOSE_Option | 7个 | 鼻子调整（上下左右、前后位置等） |
| EAR_Option | 6个 | 耳朵调整（精灵耳、大小等） |
| JAW_Option | 14个 | 下巴调整（圆度、下颚形状、长度、锐利度等） |
| SHRINK_Option | 2个 | 面部部位缩小（颈部、耳朵） |
| MMD | 47个 | MikuMikuDance兼容表情（a、i、u、e、o、微笑、眨眼等） |
| 更新用Empty | 183个 | 更新用预留形态键 |

每个类别不仅包括双眼用，还准备了左眼（_L）、右眼（_R）用的形态键，可以创建左右不对称的表情。

### 身体形态键（30个）

主要用于身体各部位的缩小设置。可以调整颈部、肩膀、手臂、手、手指、胸部、背部、腰部、腿部、脚等部位的缩小比例。

### 头发形态键（共50个）

Hair_Bangs对象包含以下功能性形态键（不包括Empty形态键）：

#### BANGS_Option（前发）- 31个
- 基本样式：
  - BANGS_OP_短く_Short：使前发变短
  - BANGS_OP_長く_Long：使前发变长
  - BANGS_OP_ずらす_Shift：使前发向侧面偏移
  - BANGS_OP_ねぐせ_Bedhair：创建睡醒发型
  - BANGS_OP_ぱっつん_Straight：创建齐刘海
  - BANGS_OP_ぱっつん短い_StraightShort：创建短齐刘海
  - BANGS_OP_ぱっつん長め_StraightLong：创建长齐刘海
- 个别调整：
  - BANGS_OP_01_Long至BANGS_OP_12_Long：可以逐条调整前发的单独发束的形态键。每个数字对应前发的特定一束，可以独立移动或改变各自的形状。
  - BANGS_OP_01_Short至BANGS_OP_12_Short：可以逐条调整前发的单独发束的形态键。每个数字对应前发的特定一束，可以独立移动或改变各自的形状。

#### FRONT_Option（前部发束）- 5个
- 个别调整：
  - FRONT_OP_01_Long：可以单独调整前部发束特定一束的形态键。
  - FRONT_OP_02_Long：可以单独调整前部发束另一束的形态键。
  - FRONT_OP_01_Short：可以单独调整前部发束特定一束的形态键。
  - FRONT_OP_02_Short：可以单独调整前部发束另一束的形态键。
  - FRONT_OP_Empty：空调整项

#### SIDES_Option（侧发）- 14个
- 基本样式：
  - SIDES_OP_短く_Short：使侧发变短
  - SIDES_OP_長く_Long：使侧发变长
  - SIDES_OP_ぱっつん_Straight：创建直侧发
- 个别调整：
  - SIDES_OP_01_Long至SIDES_OP_05_Long：可以逐条调整侧发的单独发束的形态键。每个数字对应侧发的特定一束，可以独立移动或改变各自的形状。
  - SIDES_OP_01_Short至SIDES_OP_05_Short：可以逐条调整侧发的单独发束的形态键。每个数字对应侧发的特定一束，可以独立移动或改变各自的形状。
  - SIDES_OP_Empty：空调整项

通过使用这些带数字的形态键（01、02等），可以自由调整特定发束，而不是整体发型。例如，只移动前发左侧的特定一束，或者只改变侧发的特定一束的形状。这样可以根据自己的喜好定制发型的细节。

## 关于Empty形态键

在Unity中更新时，插入新形态键可能会使现有形态键的顺序发生变化，导致与已设置的数值不一致。为防止这种情况，预先放置了多个空形态键（Empty）。更新时，这些Empty形态键将用于添加新功能。

## 形态键使用方法

1. 在Unity检视器中选择模型
2. 选择SkinnedMeshRenderer组件
3. 在"Blend Shapes"部分调整各形态键滑块
4. 设置0至100之间的值（0=无效果，100=最大效果）

通过组合形态键，可以创建自己独特的表情和体型。

## 形态键调整注意事项

### 对动画的影响

对眼睛和嘴巴等与表情相关的部分进行大幅度更改时，默认表情动画可能无法正常工作。这是因为动画是基于原始形状创建的。

例如：
- 大幅度改变眼睛大小或位置会使眨眼动画变得不自然
- 改变嘴部形状会导致说话动画不匹配
- 改变脸部比例可能会破坏整个表情动画

在这种情况下，需要调整动画片段以适应新形状。

### 形态键的灵活应用

在Menno模型中，可以将表情用形态键（EYE、MOUTH、EYEBROW等）用于体型调整，反之，也可以将详细调整用的形态键（_Option类别）用作表情。

例如：
- 组合EYE_Option系列形态键创建独特的眼部表情
- 使用MOUTH_Option创建特殊的嘴部形状作为表情
- 将通常用于调整的JAW_Option融入表情动画中

### 表情动画调整方法

如果大幅度改变了形态键，也需要调整表情动画。在Menno模型中，可以通过以下步骤进行调整：

1. 在Animator组件中设置Menno_Layer_FX.controller
   - 选择Menno虚拟形象的根对象
   - 向Animator组件添加"Menno_Layer_FX.controller"
   - 此控制器包含表情动画的默认设置

2. 在Animation窗口中调整默认表情
   - 从Unity的"Window"→"Animation"→"Animation"打开Animation窗口
   - 在场景中选择Menno虚拟形象
   - 选择Animation窗口中显示的现有表情动画
   - 更改与已调整形态键对应的关键帧
     - 点击关键帧编辑数值

3. 测试调整后的表情
   - 使用VRChat Gesture Manager等检查表情的动作
   - 根据需要进行微调

通过这种方法，可以适当调整现有表情动画以匹配自定义的形态键设置。
